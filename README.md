## About 
 - ToDo list web-app built with .NET Blazor.
 - Blazor WASM app
- .NET version:  7.0.11
- Visit [site](https://todolist-client.pages.dev/)
## Features
- Task operations such as: *add, remove, status*
- Pagination
- Sorting of tasks
- Browser local storage for storage management
- Bootstrap for UI
